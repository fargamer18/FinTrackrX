# Next.js Backup

This folder contains the original Next.js version of FinTrackrX before migration to Spring Boot.

## Contents

- `app/` - Next.js App Router pages and API routes
- `components/` - React components
- `lib/` - Utility functions and database connections
- `types/` - TypeScript type definitions
- `tests/` - Test files
- `public/` - Static assets
- `scripts/` - Node.js scripts
- Configuration files: `package.json`, `next.config.ts`, `tsconfig.json`, etc.

## To Run the Next.js Version

1. Install dependencies:
   ```bash
   npm install
   ```

2. Set up environment variables (create `.env.local`)

3. Run development server:
   ```bash
   npm run dev
   ```

## Migration Date

This backup was created on November 13, 2025, when the project was migrated to Spring Boot.

The new Spring Boot version is in the root directory.
